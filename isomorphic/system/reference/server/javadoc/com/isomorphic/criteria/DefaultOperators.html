com.isomorphic.criteria

Class DefaultOperators

  • java.lang.Object
    • com.isomorphic.criteria.DefaultOperators

  • public class DefaultOperators
extends java.lang.Object
    Helper class for accessing operators that needs for working with Criteria. Also it contains set of methods for values check.

    You can use any operator directly like DefaultOperators.Equals or you can get all of them using DefaultOperators.getDefaultOperatorMap() method.
